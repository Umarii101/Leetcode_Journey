🗺️ The Roadmap
Phase 1 — Foundation (Weeks 1–4)
Before touching hard problems, you need to be solid on the basics. Focus on these topics in order:

Arrays & Strings — the bread and butter of every interview
Hashing (HashMaps, HashSets) — unlocks O(n) solutions for many array problems
Two Pointers — elegant technique for sorted arrays and string problems
Sliding Window — essential for subarray/substring problems

Stick to Easy problems only during this phase.

Phase 2 — Core Data Structures (Weeks 5–10)

Linked Lists — reversal, fast/slow pointers
Stacks & Queues — monotonic stacks are surprisingly powerful
Binary Search — not just searching, but a thinking pattern
Trees (Binary Trees, BSTs) — recursion clicks here
Heaps / Priority Queues — top-K problems, scheduling

Mix Easy and Medium problems here.

Phase 3 — Algorithms (Weeks 11–18)

Recursion & Backtracking — subsets, permutations, combinations
Graphs — BFS, DFS, connected components
Dynamic Programming — the hardest but most rewarding topic

Start with 1D DP (climbing stairs, house robber)
Then 2D DP (grids, strings)


Greedy — knowing when to be greedy vs. DP

Mostly Medium, start touching Hard problems in DP/Graphs.

Phase 4 — Advanced (Week 19+)

Tries
Segment Trees / BIT
Advanced Graph (Dijkstra, Union-Find, Topological Sort)
Bit Manipulation


📅 Daily Routine
A good structure for each day's session:

Read the problem carefully (5 min) — don't rush to code
Think out loud / on paper before typing anything (10 min)
Code your solution
If stuck for more than 20–25 minutes, look at the hint or approach — not the full solution
After solving (or reading the approach), understand the time & space complexity
Read a top-rated discussion solution to see how others think


🔢 Recommended Starting List
Start with the classic "NeetCode 150" — it's one of the most well-curated problem lists that follows exactly this topic progression. You can find it at neetcode.io. It's completely free and has video explanations for every problem.
